# UdemyEdu
An education app that uses Udemy Apis to make clients track the contents of the courses.

#### Developed with:
- Kotlin
- MVVM architecture
- Android architecture components
- Navigation Jetpack
- Room Database
- Single activity
- dataBinding
- viewBinding
- Coil
- Coroutines
- Retrofit
- Repository pattern
